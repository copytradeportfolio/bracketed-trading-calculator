# 📈 Bracketed Trading Growth Calculator

A Streamlit web app that simulates compound account growth for copy traders using a bracket-based lot scaling strategy.

## Features
- Compound daily profit simulation
- Bracket-based lot size progression
- Interactive Plotly chart with bracket-up markers
- Day-by-day breakdown table
- Dark modern UI

## Default Settings
| Parameter | Default |
|-----------|---------|
| Starting Balance | $100 |
| Net Daily Profit | 3% |
| Bracket Size | $100 |

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
1. Fork/push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub and select this repo
4. Click **Deploy**
